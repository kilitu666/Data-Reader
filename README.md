# Data-Reader
Read and manage data. Output data as regulated style
## Description
This project was completed as part of FIT9136 (Programming Foundations) at Monash University.  
The assignment focuses on developing a Python program that applies fundamental programming concepts to solve a structured problem.

The project demonstrates the use of control flow, functions, data structures, and file handling to implement a clear and logical solution.

---

## Project Objectives
- Apply core Python programming concepts
- Use functions to structure reusable code
- Work with data structures such as lists and dictionaries
- Implement file input and output operations
- Develop logical problem-solving and debugging skills

---

## Technologies Used
- Python
- Command-line execution
- Basic file handling
